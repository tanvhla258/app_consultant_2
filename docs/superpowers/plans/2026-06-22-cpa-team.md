# CPA Team Section Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement section 07 "Meet Our Expert CPA Team" — a left-aligned headline + right-side subhead, followed by a 3-card credentials grid that reuses the 3 Partners through a credentials-focused lens (distinct from `AboutSection`'s bio framing).

**Architecture:** Three presentational components (`TeamSection`, `TeamHeadline`, `TeamMemberCard`) backed by `data/team.ts` which **imports `PARTNERS` from `components/about/data/partners.ts`** (single source of truth for partner identity) and layers credential-grid-specific fields on top.

**Tech Stack:** Next.js 16, React 19, TailwindCSS v4, TypeScript, `next/image` for partner photos, existing `useLang()` hook from `lib/i18n/`.

**Note on verification:** No test runner in this project (per `CLAUDE.md`). "Verify" steps use `npm run lint`, `npx tsc --noEmit`, and manual visual check via `npm run dev` at `http://localhost:3000`.
